🔌 WebUSB & Web Serial Printer Port Discoverer

Standard web browsers cannot access the operating system's installed printer list (like the Windows Spooler) directly due to sandboxed security rules. However, browsers can connect directly to hardware interfaces like WebUSB and Web Serial to communicate with thermal POS receipt printers.
